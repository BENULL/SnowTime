# SnowTime 卡牌设计哲学

## "Frostborne Elegance" — 霜华瑰宝

### 核心理念

这是一套关于**冰雪与珍宝**的视觉语言。卡牌如同从永恒冻土中发掘的古老卷轴，每一张都承载着神秘与珍贵。设计强调**层次感**与**材质表现**，让卡牌看起来像是被霜雪覆盖的宝石，值得珍藏。

### 视觉特征

**材质层次**
- 金属边框：精致的双色金属质感边框，顶部/底部使用浅色高光，左右使用深色阴影
- 角色图：作为卡牌的核心视觉元素，占据 40% 左侧空间，略带霜雾朦胧效果
- 渐变背景：深邃的渐变背景，从深紫渐变到近黑，模拟夜空与深渊

**色彩系统**
- 主色：玩家专属色彩（红/青/黄/薄荷/粉）
- 辅色：金色作为数字和装饰线条的强调色
- 背景色：深紫 `#1a0a2e` 到 近黑 `#0a0612` 渐变
- 特效色：冰蓝 `#67e8f9` 用于暴风雪牌

**排版艺术**
- 数字：使用 Cinzel 衬线字体，金色渐变，带微妙发光阴影
- 类型标签：小号大写字母，金色，位置顶部偏下
- 装饰线：玩家色彩的金色细线，分隔角色图与信息区

### 构图法则

```
┌────────────────────────────────────────┐
│ ═══════════════ (金色装饰线) ═══════════════ │  ← 玩家色渐变
│ ┌──────────────┐  ════════════════════│
│ │              │  ══ LEVEL 3 ══      │  ← 类型标签
│ │              │                      │
│ │   角色图     │       ◆              │  ← 玩家色宝石装饰
│ │  (带霜雾)    │                      │
│ │              │       7               │  ← 金色大数字
│ │              │                      │
│ └──────────────┘  ════════════════════│
│ ═══════════════ (金色装饰线) ═══════════════ │
└────────────────────────────────────────┘
```

### 细节打磨

**边框工艺**
- 外框：2px 玩家色渐变边框（金色 + 玩家色混合）
- 内框：1px 深色描边，增加立体感
- 圆角：12px，优雅但不臃肿

**角色图效果**
- 左侧 40% 宽度展示角色图
- 添加细微的霜雾遮罩（渐变透明）
- 边缘羽化效果，与背景融合自然

**数字表现**
- Cinzel 字体，Bold 700
- 金色渐变填充（#fff7cc → #ffd700 → #b8860b）
- 文字阴影：多层叠加营造立体感
- 微妙发光：`text-shadow: 0 0 20px currentColor`

**特殊牌**
- 治疗牌：绿色主调，💚 图标带光晕
- 守望牌：紫色主调，👁️ 图案
- 暴风雪牌：冰蓝色调，雪花装饰元素
