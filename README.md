# SnowTime 冰雪奇缘桌游

一款支持3-5人在线联机的Web端桌游，基于冬日冰雪主题设计。

## 游戏特色

- 🎲 **经典桌游体验**: 掷骰子、出牌、战斗结算的核心玩法
- ❄️ **冰雪主题设计**: 7层神圣枯树、果实收集、战斗计分
- 🎴 **特殊卡牌**: Healer(治疗者)、Watcher(守望者)、Blizzard(暴风雪)
- 👥 **多人联机**: 支持3-5名玩家同时在线对战
- 📱 **响应式设计**: 适配PC和移动端

## 技术栈

## 游戏规则

### 游戏组件
- **神圣树**: 7层结构（Level 1-7），Level 7永不放置果实
- **计分轨道**: 30格，含奖励格（🟡水果 ❄️战斗 🔵法力）
- **果实**: 共15个粉色果实标记
- **手牌**: 每位玩家10张牌（7张角色牌1-7 + 3张特殊牌）

### 回合流程

#### 1. 掷骰子放置果实
- 掷2个六面骰，在对应层级放置果实
- 骰子1-6对应Level 1-6，Level 7永远不会放置果实

#### 2. 出牌阶段
- 所有玩家同时选择一张牌打出
- **角色牌(1-7)**: 放置到对应层级（数值=层级）
- **Healer**: 从弃牌堆回收2张牌
- **Watcher**: 观看他人出牌后，可额外出一张牌
- **Blizzard**: 丢弃所有角色牌，获得被丢弃牌数量的分数

#### 3. 结算阶段（从上到下 Level 7→1）

**特殊牌结算**: 先结算Watcher、Blizzard、Healer效果

**战斗结算**:
- **同层冲突**: 同一层有多个角色，全部击败（每人获得`击败人数`战斗分）
- **上下击落**: 孤独上层击败紧邻下层（获得1战斗分）

**收集果实**: 结算后，每层孤独角色收集所在层果实

**法力积分**: 本回合最低位置角色获得1法力分

**奖励格判定**: 停留在奖励格且本回合获得对应分数类型，额外前进2格

#### 4. 回收阶段
- 树上角色回收至手牌
- 被击败角色进入弃牌堆

### 游戏结束
当任意玩家到达计分轨道终点时游戏结束：
1. 位置最高（树上层级最高）的玩家获胜
2. 若平局，战斗分高者获胜
3. 若仍平局，总位置高者获胜
